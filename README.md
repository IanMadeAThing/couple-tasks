# 💑 Couple Tasks

A real-time shared task management app built for couples. One person speaks or types a task, the other sees it instantly. Built with React, Firebase, and the Claude AI API.

---

## What it does

- 🎤 **Voice input** — speak a task and it gets added automatically
- 🤖 **AI parsing** — Claude AI reads your voice note and categorizes tasks into Now, Later, or Tomorrow
- 🔄 **Real-time sync** — both partners see the same task list instantly via Firebase
- ✅ **Task confirmation** — check off a task and confirm it's done with a popup
- 📅 **Completed history** — all finished tasks are logged by time (Last Hour, Today, This Week, etc.)
- 🔐 **Authentication** — each partner has their own secure login

---

## Tech stack

| Technology | Purpose |
|---|---|
| React + Vite | Frontend framework |
| Firebase Auth | User authentication |
| Firebase Firestore | Real-time database |
| Claude AI API | Voice note parsing and task categorization |
| Web Speech API | Browser-based voice recognition |
